This is a simulation of Wolfram’s Elementary Cellular Automata implemented with processing.js. It takes advantage of the fact that each rule can be represented as an 8-bit binary number. Here’s the code which determines the next generation:

``` java int elementary_cellular_automaton(rule_number, pattern) { rule = binary(rule_number, 8); return rule[7 - unbinary("00000" + pattern)]; } ```
